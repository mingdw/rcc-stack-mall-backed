# 蓝宝石代币质押商城系统

蓝宝石代币质押商城系统是一个基于区块链技术的代币质押商城系统，用户可以通过质押代币来获取代币，并且通过代币可以兑换商场里面的虚拟商品，交易方便快捷，安全可靠。

[前端源码](https://github.com/mingdw/rcc-stake-fe-react.git)

![特别鸣谢：Nunu](https://github.com/go-nunu/nunu/blob/main/.github/assets/banner.png)

### 商品规格属性设计
    - 商品属性设计 （基本属性过滤商品、销售属性展示共同属组或者属性相关信息、规格属性确定SKU的类别）
        - 商品属性分为基本属性、销售属性、其它属性，其中基本属性为分类目录的基本属性，所有商品必须包含一级、二级目录、三级目录的的属性，数据以json保存（比如NFT下面的数字艺术品下面的潮流艺术画，那么基本属性则为：{应用领域:潮流艺术画,技术形式：区块链数字艺术品,风格:简约}）
        销售属性
            - 销售属性分为价格属性、库存属性、其它属性，其中价格属性为商品的价格，库存属性为商品的库存，其它属性为商品的其它属性，数据以json保存（比如NFT下面的数字艺术品下面的潮流艺术画，那么销售属性则为：{价格:100,库存:100,其它:{}）
